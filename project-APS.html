Automatic Parking System

2024 Spring - 2024 Fall

Github Link

Project Overview

This project is to create an automatic parking system for a public parking lot. Our goals in this project are empty parking spot indication lights, car license plate recognition at the entry and exit to keep a record of the parking time in the database and an online payment system.

It is an ongoing project from 2024 Spring to 2024 Fall semesters. The contents here will be updated according to the progress of this project. Currently, we are working on project proposal and the circuit for car detection in each parking spot, using ifrared sensors.
