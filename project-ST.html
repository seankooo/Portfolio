Project Overview

This project was a junior project for one semester that we created a Space Invader clone game.
First, we designed an ALU that works as Mips assembly language on FPGA in Verilog, then we created VGA control to show game on VGA monitor. WE have created other modules for details of the game. More details about the code can be found in the Github.
