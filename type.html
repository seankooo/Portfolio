How to customize typing script

1. You create a '.txt' file of your desired script using Notepad app.

Make sure the file format you created is in '.txt' format.

2. Upload the file using the very left button

3. After uploading the file, click the process file button

4. You can practice the script sentences by sentences and move to different sentences using Prev or Next buttons.
